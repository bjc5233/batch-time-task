# batch-time-task
> batch快捷创建计划任务


### 说明
代码本质上是调用SCHTASKS命令, 生成的任务可以在[控制面板->计划任务]中查看或配置参数

|script|name|
|-|-|
|taskEveryNDay.bat|每日计划任务|
|taskEveryNHour.bat|小时间隔计划任务|
|taskEveryNMinute.bat|分钟间隔计划任务|
|taskOnce.bat|一次性计划任务|
